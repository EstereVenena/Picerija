# :pizza: Picerija
:fork_and_knife: Programma kas ļauj izveidot picas pasūtījumu!

## Programmas darbība
![pica](https://github.com/EstereVenena/Picerija/assets/165994277/b8603544-b931-4494-bb24-de3c5ad017b1)
